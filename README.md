#     SSM集成的项目，项目使用Maven管理

▇ 项目使用Spring4.1.2.RELEASE + SpringMVC4.1.2.RELEASE + Mybatis3.3.0

▇ 项目集成了Mybatis分页插件  

▇ 项目使用的mysql数据库，在Linux系统管理数据库。

▇ 运用spring4 对泛型Mapper 的注入

▇ 前端 使用 Bootstrap + Jquery + CSS + Jsp + Sco + Angular

▇ 服务器 Tomcat7 

▇ 项目使用 redis 提高系统性能
